# 程序目录结构
## conf
> 程序配置信息 settings.py
## log
> 程序执行日志,以log为一级目录,日期格式为二级目录,加载csv文件名.log为日志文件
## utils
> 程序使用工具类
>execl.py   execl相关函数
>fileParse.py   文件解析相关函数
>ftp.py ftp相关函数
>hive.py   hive相关函数
>log.py 日志相关函数
>mysql.py   mysql相关函数
## LocalCSVToHive.py
> csv文件ftp加载及hive入库主程序
